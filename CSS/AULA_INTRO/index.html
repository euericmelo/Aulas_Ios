Aula CSS

CSS (Cascading Style Sheets ou Folhas de Estilo em Cascata) é linguagem de estilo ultilizada para descrever a apresentaçao de documentos escritos em HTML ou XML. Ela define a aparência visual de uma pagina web, controlando os elementos como cores, fontes, espaçamentos, layot e design.

Existem 3 formas de aplicar CSS

INLINE - Na mesma linha.

INTERNO - Dentro do documento html, com a tag style.

EXTERNO - Dentro de um outro arquico .css e ultilizando seletores.
